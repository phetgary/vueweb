# 電子商務網站
## 專案介紹 
##### 使用工具
* Javascript
* CSS/HTML/Bootstrap4
* Vue.js
* Vue router/ Vue Cli
* 使用AJAX取得第三方API資料

### 1. 這是一個電子商務的網站 
### 會員登入後可在後台新增編輯及管理訂單、優惠券代碼
### 使用Vue Cli/Webpack 開發此專案
### 後台資料為串接第三方API
### 使用Vue Router 建置網站路由
### 套用Bootstrap4排版及部分手刻頁面
